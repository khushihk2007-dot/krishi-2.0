# 🌾 Krishi

Krishi is a smart agriculture ecosystem platform designed to digitally connect **Farmers, Buyers, and Labourers** through one unified system.

The platform helps farmers sell crops directly, hire labour, access market insights, discover government schemes, rent equipment, and make data-driven farming decisions.

---

# 🚀 Features

## 👨‍🌾 Farmer Module

* Crop listing marketplace
* Smart field intelligence dashboard
* Weather and market insights
* Equipment rental system
* Government schemes section
* FPO discovery
* Real-time farming updates

## 🛒 Buyer Module

* Browse live crop listings
* Compare mandi prices
* Place crop orders
* Export-oriented marketplace
* Search and filter crops
* Direct farmer-to-buyer interaction

## 👷 Labourer Module

* Labour job marketplace
* Crew hiring system
* Daily wage listings
* Availability tracking
* Real-time job applications
